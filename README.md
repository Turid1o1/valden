# VALDEN

Внутренний репозиторий платформы удалённого доступа VALDEN.

## Правовой статус

Все материалы репозитория (исходный код, графика, тексты, архитектурные решения, бренд-элементы) являются объектами авторского права и охраняются законом.

Использование регулируется файлом `LICENSE`.

## Назначение репозитория

- внутренняя разработка платформы;
- инженерные проверки и тесты;
- подготовка релизов для официального канала дистрибуции правообладателя.

## Дистрибуция продукта

Публичная установка, сборка и распространение продукта из этого репозитория не предусмотрены.

Клиентские дистрибутивы публикуются правообладателем через официальный канал.

## Структура

- `backend/` — серверные компоненты;
- `client/` — desktop-клиент;
- `site/` — веб-часть;
- `infra/` — инфраструктурные шаблоны;
- `scripts/` — внутренние утилиты.
